# 🧙 JavaScript RPG Game

A simple **Role Playing Game (RPG)** built with **Vanilla JavaScript**, HTML and CSS as part of the JavaScript curriculum from FreeCodeCamp.

The game demonstrates core JavaScript concepts such as **DOM manipulation, game state management, arrays, objects, and event handling**.

---

## 🎮 Game Concept

The player explores a small world, fights monsters, earns gold and experience, and upgrades their weapons in the store.

The goal is to survive battles and become strong enough to defeat stronger enemies.

---

## ⚙️ Features

* ⚔️ Fight monsters
* 🏪 Buy health and weapons in the store
* 💰 Earn gold after winning battles
* 🧠 Randomized enemy attacks
* 📈 Player progression (health, XP, weapons)

---

## 🧠 JavaScript Concepts Used

This project focuses on practicing fundamental JavaScript skills:

* DOM manipulation
* Event listeners
* Arrays and objects
* Game state management
* Conditional logic
* Random number generation
* Functions and modular logic

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/javascript-rpg-game.git
```

2. Open the project folder

```
cd javascript-rpg-game
```

3. Open `index.html` in your browser.

No installation or dependencies required.
